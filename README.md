## 📋 Description

TOKOSSA est une plateforme de services de livraison rapide au Bénin. L'application permet aux utilisateurs de commander des services flash, suivre leurs livraisons et gérer leur compte. Interface moderne et intuitive optimisée pour le marché béninois.

## 📁 Structure du projet

```
TOKOSSA/
├── src/
│   ├── components/     # Composants UI
│   ├── features/       # Modules métier
│   ├── pages/          # Pages de l'application
│   ├── services/       # API et services
│   ├── stores/         # State management
│   └── utils/          # Utilitaires
├── server/             # Backend
├── scripts/            # Scripts utilitaires
├── package.json
├── tailwind.config.js
└── vite.config.ts
```

## 🛠️ Technologies

- **TypeScript** — Typage statique
- **React** — Interface utilisateur
- **Vite** — Build tool rapide
- **Tailwind CSS** — Styling utilitaire

## ✨ Fonctionnalités

- Commande de services de livraison en ligne
- Système d'authentification
- Dashboard utilisateur
- Interface responsive mobile-first
- Suivi de commandes
